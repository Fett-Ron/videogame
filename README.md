# zombiE gunN
### Marcelo Picado Leiva, C15950

es un videojuego top-down de matar zombies por oleadas  
cuando mate a todos los zombies avanzará a la siguiente oleada en la que encontrará más zombies,  por el momento mi record es de 30 kills.  

Se puede mover con WASD, disparar con la J y ponerle pausa con la P
